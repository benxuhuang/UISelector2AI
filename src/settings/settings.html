💡 How it works
🎙️ Speech-to-Text (STT) — Records your voice and transcribes it into text. Choose a provider that supports audio transcription. Groq uses the standard whisper endpoint (/audio/transcriptions). OpenRouter routes via chat completions with input_audio. Custom lets you enter any OpenAI-compatible URL; the transport (audio endpoint vs chat completions) is auto-detected.
🤖 LLM Refiner — After transcription, the text is sent to an LLM to fix punctuation, capitalization, and preserve technical terms. The default system prompt is tailored for web development annotations. Disable it if you want the raw transcription.
🌐 Language — Leave empty for auto-detect. Set an ISO code (e.g. en, es) to force a specific language for better accuracy with accents or mixed-language dictation.
📋 Workflow — Click the mic button 🎤 in any annotation modal → button turns red (starting) → turns green (ready, speak now) → click again to transcribe → text appears in the input. Use Test STT and Test LLM buttons to verify each step.
